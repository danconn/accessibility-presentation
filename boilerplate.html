What is accessibility?

Accessibility includes but is not limited to Section 508 and is defined as...

The inclusive practice of making websites usable by people of all abilities and disabilities.

Introduce concepts and buzzwords

WCAG

Web Content Accessibility Guidelines (WCAG) - "wih-cag"
A set of stepped guidelines created by the W3C for authoring accessible Web content.
Section 508 standards were based on WCAG 1.0 and revisions will be based on WCAG 2.0 AA.

Introduce concepts and buzzwords

WAI-ARIA

Web Accessibility Initiative - Accessible Rich Internet Applications (WAI-ARIA) - "ah-ree-uh"
It defines a way to make Web content and Web applications more accessible to people with disabilities.
It especially helps with dynamic content and advanced user interface controls developed with Ajax, HTML, JavaScript, and related technologies.
